# lowi
어셈블리어 문법 기반 스크립트 언어

## 특징
- 실제 어셈블리어를 거의 수정 없이 사용할 수 있습니다.
- 확장성이 높습니다. 직접 아키텍쳐를 정의할 수도 있습니다.

## lowi-lite
lowi에서 확장 관련 기능을 모두 제거해 lowi 표준 아키텍쳐에서 제공하는 기능만을 제공하는 lowi의 서브 버전입니다. lowi보다 빠르고 가볍습니다.<br>
**아직 준비중입니다.**

## lowi-interpreter
lowi 표준 아키텍쳐대로 코드를 입력하면 인터프리팅을 해 주는 프로그램입니다.<br>
**아직 준비중입니다.**
